Promise

Promise is an asyncronous way of Programming where a function executes and issues a call back. When the function execution is complete the call back method is called and is executed. For example function called to fetch data from API when data is recieved the call back is executed
Difference between Promise and Observable
  1. Promise is always Asyncronous Observable can be both
  2. Promise returns Success or failure Observable returns strams
  3. Promise can't be cancelled it will always execute and return success or failure callback. Observables can be unsubscribed
